laravel-blade-snippets
======================

Sublime 3 snippets for Laravel blade files

##Usage
|  Trigger   	|  Content                                        |
|:--------------|-------------------------------------------------|
|  %Form     	|  `{{ Form::open() }}`                           |
|  %label    	|  `{{ Form::label('', '') }}`                    |
|  %text     	|  `{{ Form::text('') }}`                         |
|  %password 	|  `{{ Form::password('', array('id'=>'')) }}`    |
|  %hidden   	|  `{{ Form::hidden('', '') }}`                   |
|  %radio    	|  `{{ Form::radio('', '') }}`                    |
|  %checkbox 	|  `{{ Form::checkbox('', '') }}`                 |
|  %file     	|  `{{ Form::file('') }}`                         |
|  %email    	|  `{{ Form::email('', null, array('id'=>'')) }}` |
|  %textarea 	|  `{{ Form::textarea('') }}`                     |
|  %submit   	|  `{{ Form::submit() }}`                         |
